JavaScript

JavaScript within the Page

I have used minimal additional (read my own) JavaScript within the page, as the majority of the functionality I’ve added has JavaScript already written for it in the bootstrap library.

The specification states:

Integrate one or more features of JavaScript into your site to make your site more interactive. For example, you can use JavaScript to add alerts, to have an effect at a recurring interval, or to add interactivity to buttons, dropdowns, or forms. Feel free to be creative!

To meet this specification, I added some custom JavaScript to automatically cycle through the carousel I added. There’s an option that you can implement this functionality directly into the class tag, but taking it out meant I needed script for it.

I used the following code to meet this specification:

        
      $('#carouselOne').carousel({
        interval: 3000,
        cycle: true,
        pause: "null"
      })

This code sets the interval that the carosel dwells on an image to 3000ms, and prevents the carousel pausing on mouseover.

I may end up requiring additional JavaScript to get my navbar to work though - more on that in the Navigation Panel section.
