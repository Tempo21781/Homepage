The Basic Carousel

Placing the basic carousel was straight forward, as was filling it with images. I ran into problems when trying to automate the image transitions. My initial JavaScript code did nothing to the appearance on the screen.

Doing some research identified the cause of the issue as configuration issues with the bootstrap I was implementing. The guides I’ve been reading have all been written for Bootstrap 4, and the bootstrap I’d linked into my code (given to us by the distributed code) was bootstrap 5. The quick fix was simply to instead import bootstrap 4.0.0 instead of the 5.something I’d started with. Later in the project I may try and fix this for version 5, though I’m not sure.

Just as I finished writing this, I realised that the styling I’d added causes an issue for mobile devices. I’d set the container height to be fixed, which caused the images to squish. I needed to amend to a min-height and a max-height with overflow hidden to get what I needed.
